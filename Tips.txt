CLAUDE Commands:

/exit
Sluit de chat volledig af — alsof je de sessie beëindigt. De hele context gaat verloren.

/clear
Leegt de huidige sessiecontext en chatgeschiedenis. De sessie blijft open, maar zonder context.

/compact
Maakt een samenvatting (compact summary) van het hele gesprek tot nu toe en verwijdert de rest van de context.
Je behoudt de belangrijkste informatie in beknopte vorm. Zeer nuttig om lange sessies “licht” te houden.

2 keer ESC
Laat je terugspoelen naar een eerder punt in de sessie (revert). Je kunt daarna verder werken vanaf dat punt
Gebruik wanneer:
Je merkt dat Claude een verkeerde richting is ingeslagen in de context of verkeerde aannames heeft gemaakt.

/resume
Herlaadt de laatste sessie (ook als je Claude of de terminal hebt afgesloten).
Herstelt het gesprek met de vorige context.
Gebruik wanneer:
Je een project later opnieuw wilt oppakken, zonder alles opnieuw te moeten uitleggen.


CONTEXT WINDOW:

Het context window bepaalt hoeveel informatie het AI-model tegelijk kan “onthouden” of verwerken tijdens een sessie.
Je kunt het zien als het werkgeheugen van het model.

Now the context window is about 200000 tokens in Claude Code at this time.
But if your sessions last a long time, have a lot of manually added context and back and forths,
then that context window is eventually going to fill up.

Wanneer bij automatische deployment, nadat de data en de home pagina geseed is, maar je op de homepage / toch een 404 error krijgt.
Zorg ervoor dat je de Fabricator route cache cleared in je script!
Command hieronder:
php artisan optimize:clear && php artisan config:clear && php artisan route:clear && php artisan cache:clear
